How to Use

  • Vertical bars show the percentage of experts who selected each vulnerability level
  • Numbers below bars show the percentage and count of expert responses (e.g., "32% (8)")
  • Row height automatically adjusts based on the data range for optimal space usage
  • Median indicator (▼) marks the 50th percentile vulnerability assessment
  • URL parameters allow switching between different risk scenarios (?risk=1.1, ?risk=1.2, etc.)
